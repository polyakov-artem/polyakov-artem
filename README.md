### Обо мне

С энтузиазмом изучаю все, что связано с веб-разработкой. Все свободное время уделяю развитию в этом направлении. Цель — заниматься любимом делом, постоянно улучшать свои технические навыки и этим способствовать успеху команды. В данный момент выполняю задачи из курса по React от RSS School.

Личные качества: отзывчивый, спокойный, ответственный, целеустремленный.

Хобби: волейбол, сноуборд.
